# Simple CLI Calculator

A simple and interactive Command Line calculator developed in C# and .NET.

I made this project just to practice my C# skills.

## Features

It can perform addition, subtraction, division, modulo, and find the roots of quadratic functions.

## Technologies Used

* **Language:** C#
* **Platform:** .NET (Console Application)
* **IDE:** VS Code
